# Task-1-IbrahimAdekunle

## Project Overview

This project demonstrates the data cleaning and preparation of an e-commerce sales dataset using Microsoft Excel. The objective was to improve data quality and prepare the dataset for exploratory data analysis by ensuring its accuracy, consistency, and reliability.

## Project Objectives

* Prepare the dataset for analysis.
* Identify and treat missing values.
* Verify duplicate records and Order IDs.
* Standardise text fields.
* Validate date, numerical, and categorical data formats.
* Improve overall data consistency and reliability.

## Data Cleaning Activities

* Converted the dataset into an Excel Table for efficient data management.
* Replaced missing CouponCode values with **"NoCouponCode"**.
* Verified duplicate records and duplicate Order IDs.
* Applied text standardisation techniques where appropriate.
* Validated data formats and checked for inconsistencies.
* Performed final quality checks to ensure data integrity.

## Tools Used

* Microsoft Excel
* Excel Tables
* Data Validation
* Remove Duplicates
* TRIM
* PROPER

- Cleaned_Dataset.xlsx  Dataset after data cleaning and preparation.
